HangMan-Game

Hangman is a classic word guessing game built using JavaScript. Players guess letters to reveal a hidden word. Wrong guesses draw parts of a stick figure. Game ends when the word is guessed or the figure is complete. Clean interface with interactive gameplay. Responsive design for all screen sizes. Includes a restart button to play again. Fun and educational for improving vocabulary.

Ideal for beginners to practice DOM manipulation and UI design.
